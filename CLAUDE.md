# CLAUDE.md

이 파일은 Claude Code가 이 프로젝트에서 따라야 할 규약을 정의합니다.

## 커밋 메시지 규약

Conventional Commits 형식을 사용합니다. 타입은 영어, 내용은 한국어로 작성합니다.

```
<type>: <변경 내용 요약>
```

### 타입 목록

| 타입 | 사용 시점 |
|------|-----------|
| `feat` | 새로운 기능 추가 |
| `fix` | 버그 수정 |
| `refactor` | 기능 변경 없이 코드 구조 개선 |
| `style` | UI/스타일 변경 (기능 영향 없음) |
| `docs` | 문서 수정 (README, 주석 등) |
| `chore` | 빌드 설정, 패키지, 기타 잡무 |

### 예시

```
feat: 프로젝트 상세 페이지 추가
fix: 이미지 슬라이드 오류 수정
refactor: ProfileContainer 구조 개선
style: 버튼 hover 스타일 수정
docs: README 갱신
chore: 패키지 업데이트
```

### 규칙

- 제목은 명령형으로 작성 (예: "추가했다" 아닌 "추가")
- 제목 끝에 마침표 없음
- 제목은 50자 이내로 간결하게
- 필요 시 본문에 변경 이유나 맥락 추가
